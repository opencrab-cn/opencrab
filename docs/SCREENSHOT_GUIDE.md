## 📐 截图尺寸规范

### 标准尺寸
- **主截图**: 1280x720 (16:9)
- **对话框截图**: 800x600 (4:3)
- **移动端适配**: 750x1334 (可选)

### 格式要求
- **文件格式**: PNG (无损压缩)
- **色彩空间**: sRGB
- **位深度**: 24-bit
- **DPI**: 72 (屏幕显示)

### 命名规范
```
{功能模块}-{具体内容}-{主题}.png

示例:
- login-page-dark.png
- chat-interface-markdown-dark.png
- plugin-market-featured-dark.png
```

## ⚠️ 敏感信息处理

### 必须打码的内容
- [ ] OAuth Client ID 和 Secret
- [ ] API Token 和 Access Key
- [ ] 用户个人账号和密码
- [ ] IP 地址和端口号
- [ ] 文件完整路径
- [ ] 邮箱地址和手机号
- [ ] 二维码和条形码

### 打码方法
```bash
# 使用 ImageMagick 模糊处理
convert input.png -blur 0x8 output.png

# 或使用在线工具
https://www.canva.com/photo-editor/
```

### 替代方案
- 使用占位符：`client_id=********`
- 使用示例数据：`token=your_token_here`
- 使用本地地址：`http://localhost:3000`

## 🛠️ 截图工具推荐

### macOS
- **系统自带**: Cmd + Shift + 4
- **Snipaste**: 免费强大，支持贴图
- **CleanShot X**: 专业截图工具

### Windows
- **系统自带**: Win + Shift + S
- **ShareX**: 开源免费，功能丰富
- **PicPick**: 界面美观，编辑方便

### Linux
- **Flameshot**: 功能强大，标注丰富
- **Shutter**: 老牌截图工具
- **系统自带**: PrintScreen

### 跨平台
- **OBS Studio**: 录屏 + 截图
- **Lightshot**: 快速分享
